Usability Reporting Manager (URM)
 ... provides an interactive Web-Interface to enter Usability Reporting Results 
 ... allows to export into UsabML XML format (see: http://itm.fh-joanneum.at/usabml)

is 
 ...a RubyOnRails WebApplication
 ...free Software (GNU GPLv3, http://www.gnu.org/licenses/gpl.txt)

test it
 ...with the free online demo at: http://urm.feiner.at

install and usage
 ... checkout from (git) repository https://github.com/internettechnik/urm
 ... setup and configure (e.g. mysql) database
 ... run ruby on rails application (on standalone server or e.g. on apache)
 ... use any modern browser to visit the web page

(by internettechnik, 2010 http://itm.fh-joanneum.at/usabml/urm )
